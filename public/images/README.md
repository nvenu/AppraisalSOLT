# Images Folder

## Logo Instructions

Place your Source of Life Technologies logo here:

- **Filename**: `solt-logo.png` or `solt-logo.svg`
- **Recommended size**: 200x60 pixels (or similar aspect ratio)
- **Format**: PNG with transparent background or SVG

The logo will appear in:
- Footer of all pages
- Login page header

## Current Files

- Place `solt-logo.png` here for the company logo
- You can also use `solt-logo.svg` for better quality

## Example

```
public/
  images/
    solt-logo.png  ← Your logo here
    README.md      ← This file
```
